# **App Name**: OptiStock Simulator

## Core Features:

- User Authentication: Simulate user sign-in and sign-up using a local JSON 'database'.
- Interactive Warehouse Designer: Enable users to design warehouse layouts using a grid-based system with tools for shelves, bays, and more, persisting layouts in local storage.
- Order Generation: Create order item lists based on the designed warehouse layouts.
- Warehouse Simulation Game: Simulate warehouse operations with modes for picking and stocking, tracking time, moves, and costs.
- Real-time Performance Metrics: Display a live dashboard of game metrics, including time, moves, and accumulated costs, to provide immediate feedback to the player.
- Actionable Insights: Analyzes a completed game in order to generate feedback to the user, in the form of specific guidance on the user's performance.
- Leaderboard and Performance History: Showcase the player's operational history using area graphs for cost and tabular data of each game; data is persisted in local storage.

## Style Guidelines:

- Primary color: Slate blue (#7395AE) to suggest precision and analysis in an industrial context.
- Background color: Light gray (#D6D5D4), of similar hue but desaturated and light, creating a neutral backdrop.
- Accent color: Dusty lavender (#9977B7) to bring some creative flourish while remaining harmonious.
- Body and headline font: 'Inter', a sans-serif typeface that balances a contemporary appearance with neutrality.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, line-based icons for warehouse elements.
- Incorporate subtle transitions and animations for user interactions, such as highlighting active elements or providing feedback on actions.
